About This App

This task manager demonstrates a complete CRUD application using SimpleRDBMS, a relational database management system built from scratch in Python.

  • ✅ Custom SQL parser and query engine
  • ✅ B-tree indexing for primary and unique keys
  • ✅ Support for INTEGER, TEXT, REAL, and BOOLEAN data types
  • ✅ CRUD operations (Create, Read, Update, Delete)
  • ✅ WHERE clause filtering
  • ✅ Interactive REPL mode
